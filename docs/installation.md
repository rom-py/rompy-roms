# Installation

## Stable release

To install rompy-roms, run this command in your terminal:

```bash
pip install rompy_roms
```

This is the preferred method to install rompy-roms, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

The sources for rompy-roms can be downloaded from the [Github repo](https://github.com/rom-py/rompy_roms).

You can either clone the public repository:

```bash
git clone git://github.com/rom-py/rompy_roms
```

Or download the [tarball](https://github.com/rom-py/rompy_roms/tarball/master):

```bash
curl -OJL https://github.com/rom-py/rompy_roms/tarball/master
```

Once you have a copy of the source, you can install it with:

```bash
pip install -e .
```
